# 🌟 C# 基礎課程說明

---

## 📚 課程簡介

本課程從基礎到進階，協助學員建立完整的 C# Windows Forms 應用程式開發能力。課程涵蓋 Visual Studio 開發環境設置、C# 語法與控制項操作、資料庫連接與操作，以及實際系統開發實作。透過循序漸進的學習方式，從簡單的 HelloWorld 程式到完整的進貨系統開發，讓學員能夠掌握 Windows 桌面應用程式的開發技能。

---

## 🗂️ 課程規劃

| 週次  | 單元主題                        | 內容摘要                                                                                                |
| ----- | ------------------------------- | ------------------------------------------------------------------------------------------------------- |
| 第1週 | 環境安裝與專案建立              | 安裝 Visual Studio、建立/開啟基本專案、使用 TextBox、RadioButton、Button 等控制項                       |
| 第2週 | Windows Forms 控制項與運算子    | 介紹 C# 運算子（加減乘除、取餘數）、實現基本數學運算                                                    |
| 第3週 | 模運算與函式                    | 學習 Modulo（模運算）、控制項屬性方法事件、認識函式與變數、條件判斷                                     |
| 第4週 | 隨機數與事件驅動                | 製作剪刀石頭布遊戲、學習 Random 隨機數、事件驅動程式設計                                                |
| 第5週 | SQL Server 資料庫基礎           | 安裝 SQL Server 與 SSMS、理解資料庫架構、建立資料庫與資料表                                             |
| 第6週 | 通訊錄維護系統（查詢與新增）    | 使用 SSMS 連線資料庫、SQL SELECT/INSERT、C# 連接資料庫、實作查詢與新增功能                              |
| 第7週 | 通訊錄維護系統（完整功能）      | SQL UPDATE/DELETE、了解正規系統設計、完成通訊錄 CRUD 完整功能                                           |
| 第8週 | 進貨系統（DataGridView 與交易） | 使用 DataGridView 顯示與操作資料、多表資料庫設計（單頭單身結構）、SqlTransaction 交易處理、實作進貨系統 |
